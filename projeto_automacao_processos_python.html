Automação de Aprovação de Pedido de Compra

Sistema em Python para automatizar o processo de aprovação de pedidos de compra com integração a banco de dados, geração de relatórios e notificação por e-mail

Visão Geral do Projeto

Desenvolvimento de uma solução completa em Python para automatizar o processo de aprovação de Pedido de Compra. O sistema realiza consultas periódicas ao banco de dados, identifica os pedidos pendentes de aprovação, gera relatórios em Excel e notifica automaticamente os gestores responsáveis.

Tecnologias Utilizadas

Python
SQL
OpenPyXL
SMTP
Agendamento de Tarefas
Excel

Fluxo do Processo

1
Consulta ao Banco de Dados
2
Filtragem de Pedidos de Compras Pendentes
3
Geração de Relatório Excel
4
Envio de E-mail para Gestores
5
Registro em Log

Funcionalidades Principais

Consulta SQL Automatizada

Consulta personalizada ao banco de dados para extrair os pedidos pendentes de aprovação, incluindo informações críticas como prazos e valores.

Relatório em Excel

Geração automática de planilhas com colunas para status (pendente, pendente há mais de 24h), links diretos para aprovação e dados relevantes.

Notificação por E-mail

Envio automático de e-mails para os gestores responsáveis, com relatório em anexo e link direto para o portal de aprovações.

Sistema de Logs

Registro detalhado de todas as execuções para acompanhamento e auditoria do processo, armazenado no servidor.
